# Car-Racer-Road
Highway Heat is a browser-based 3D racing game demonstrating real-time WebGL rendering and procedural audio generation. Built using vanilla JavaScript and the Three.js library, it features custom collision physics, dynamic obstacle spawning, and synthesized engine sound effects.
